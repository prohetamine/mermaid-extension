Common API
You should use the .mm script as follows, forming a fetch request, you can send one or more requests for one event, also cancel requests and slow down their execution, set the number of attempts to send a request.
Script.mm
{
  "chaturbateEvent": [{
    "fetch": {
      "url": "..."
    },
    "cancel": 13000,
    "attemptsError": 30,
    "attemptsSleepError": 3000
  }],
  "bongacams": [{ ... }, { ... }],
  "myfreecams": [{ ... }]
}
You can see this example and others in the examples folder
